## Usage Guidelines

### Screenshot Standards
- **Resolution:** 1920x1080 or higher (scaled down for docs)
- **Format:** PNG (lossless for clarity)
- **Size:** Compress PNGs before committing (use TinyPNG or similar)
- **Content:** Avoid showing personal files/folders

### File Naming
Use descriptive, lowercase, hyphen-separated names:
- ✅ `drag-drop-demo.png`
- ✅ `main-window-v0.2.png`
- ❌ `Screenshot 2025-01-01.png`
- ❌ `IMG_1234.png`
